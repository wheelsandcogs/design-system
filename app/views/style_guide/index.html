Style guide

Guidance for styling internal and caseworking services in the Home Office. This includes any service that does not sit on service.gov.uk or gov.uk.

  • Typography
  • Colour
  • Header
  • Footer

How to use these styles

These styles override the default GOV.UK styles. Any styling not specified in this guide should default back to GOV.UK styles.

Home Office services that should follow these styles are:

  • caseworking systems
  • microsites
  • HR and development platforms
  • intranets
