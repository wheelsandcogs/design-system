Style guide

This style guide covers styling for internal and case working services in the Home Office. This includes services that do not sit on service.gov.uk or gov.uk.

  • Typography
  • Colour
  • Header
  • Footer

How to use these styles

The Home Office design system is based on, and draws extensively from, the GOV.UK design system, adjusted for user needs for internal services. These styles override the default GOV.UK styles. Any styling not specified in this guide uses GOV.UK styles by default.

Home Office services that should use these styles are:

  • caseworking systems
  • microsites
  • HR and development platforms
  • intranets
