Colour

Color should be used thoughtfully, have a real user need and any use backed by research.

Primary colours

#8F23B3
Purple should be used sparingly. It can be used as a keyline in headings.
#0B0C0C
Black is used for headings and body content on pages.
#F2F2F2
Light grey can be used as a main content background colour on frequent-use services.

Secondary colours

#3D5C00
#005EA5
#3B3B3B
#DADADA

Accessibility

Text and background colors need to meet the minimum contrast ratios specified by WCAG 2.0.
