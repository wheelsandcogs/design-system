Typography

GDS Transport, the typeface used on GOV.UK is restricted, by license, for use on gov.uk domains. Services for internal users cannot use New Transport. These services should instead use the open source Inter UI typeface.

Inter UI is closely related to Helvetica and Arial, but offers improved legibility and readability for digital interfaces, particularly at smaller type sizes.

Headings

Markup headings semantically using the appropriate heading level and use a corresponding heading class to apply styling.

Body content

Inter UI has a taller x-height and larger letter-spacing then GDS Transport. The default font size is 18px on large screens and 15px on smaller screens.

Multiuse caseworking services tend to be more complex and should be optimised for speed. Smaller body sizing is acceptable but the minimum font size on large screens should be 15px.

Links

Links should be blue and underlined.

Typographic scale

Scale helps to clearly define the hierarchy of your content.

Element Size Font weight
Heading-xlarge 40px bold
Heading-xlarge 32px bold
Heading-medium 24px bold
Heading-small 18px bold
Body 18px regular

Line height

Inter UI requires a different line height to GDS Transport. Line height should be set to 1.45 on larger screens and 1.35 on smaller screens.
