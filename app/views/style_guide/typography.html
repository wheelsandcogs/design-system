Typography

New Transport, the typeface used on GOV.UK is restricted, by license, for use on gov.uk domains. Services for internal users cannot use New Transport. These services should instead use the open source Inter UI typeface.

We are recommending Inter UI as a typeface as it is very closely related to Helvetica and Arial, but offers improved legibility and readability for digital interfaces, particularly at smaller type sizes.

Headings

Heading xlarge

Heading large

Heading medium

Heading small

Body content

Inter UI has a taller x-height and larger letter-spacing then New Transport. The default font size is 18px on large screens and 15px on smaller screens.

Multiuse caseworking services tend to be more complex and should be optimised for speed. Smaller body sizing is acceptable but the minimum font size on large screens should be 15px.

Links

Links should be blue and underlined.

Line height

Inter UI requires a different line height to New Transport. Line height should be set to 1.45 on larger screens and 1.35 on smaller screens.
