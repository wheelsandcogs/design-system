Typography

GDS Transport, the typeface used on GOV.UK, is restricted by license for use on gov.uk domains. Services for government users cannot use GDS Transport and should instead use the open source Inter UI typeface.

Inter UI is closely related to Helvetica and Arial, but offers improved legibility and readability for digital interfaces, particularly at smaller type sizes.

Headings

Markup headings semantically using the appropriate heading level and use a corresponding heading class to apply styling.

Body content

Inter UI has a taller x-height and larger letter-spacing then GDS Transport. The default font size is 18px on large screens and 15px on smaller screens.

Multiuse caseworking services tend to be more complex and should be optimised for speed. Smaller font sizes are acceptable but the minimum on large screens should be 15px.

Links

Links should be blue and underlined.
