Typography

Services should use Inter UI font in place of Transport. Inter UI is an open source font, specially designed for the legibility and readability of user interfaces.

Headings

Heading xlarge

Heading large

Heading medium

Heading small

Body content

Inter UI has a taller x-height and larger letter-spacing then Transport. Therefore the default font size is 18px on large screens and 15px on smaller screens.

Multiuse caseworking services tend to be more complex and should be optimised for speed. Smaller body sizing is acceptable but the minimum font size should be 15px.

Links

Links should be blue and underlined.

Line height

Inter UI requires a different line height to Transport. Line height should be set to 1.45.
